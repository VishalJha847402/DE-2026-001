# Build-in-Public Content System 🔒

> Standalone rulebook (kept separate from CLAUDE.md to keep that file lean). Authoritative — referenced by `PENDING.md`.
> Turns Vishal's learning into LinkedIn / X / YouTube / Insta content.

## Core principle
Content is a **byproduct of learning**, never a second job. Vishal studies; **Claude drafts**; Vishal adds the real human touch; Vishal posts (~10 min). This is also the **"explain it" gate** of the First Principle — teaching publicly = deep learning + career visibility.

## Golden rules
- **Proof > claims** — show real code, GitHub, the OrderIQ project/dataset. Receipts beat tips.
- **Genuine voice** — personal journey, simple English, humble *learner* (not guru), admit confusion/mistakes.
- **Never post a raw AI draft** — Claude gives substance + structure + draft; Vishal injects one real sentence (what confused/clicked today) to make it his.
- **Engage > broadcast** — ~15 min/day genuinely commenting on others grows faster than posting.
- **Consistency > perfection** — 80%-good-and-posted wins. One niche: the DE learning journey.

## 🚫 Anti-slop banned list
"In today's fast-paced world" · "Let's dive in" · "game-changer" · emoji-stuffing · corporate buzzwords · soulless listicles.

## Platform priority
1. 🥇 **LinkedIn** (recruiters live here) + 🥈 **X** — start here.
2. 🥉 **YouTube** later — "study with me" live (accountability, zero-management) + occasional edited project walkthroughs.
3. **Instagram** last — repurposed Reels.

## Format = text backbone + strategic visuals
Visuals are ~free: reuse each lesson's **Mermaid diagram** or a **code screenshot** (running on OrderIQ).

**LinkedIn post:**
`Hook (line 1, stops scroll)` → whitespace/short lines → `the story (what you learned / got stuck on)` → `the "aha" (simply)` → `the value (plain words)` → `takeaway + a question` → 3–5 tags. Attach a diagram/code image on most posts. **1 carousel/week** for a deep topic.

**X:** single punchy tip, OR a thread = `hook tweet` → 4–7 tweets (one idea each) → `takeaway`. Add a clean **code screenshot**.

## Workflow — the "Content Pack"
After each lesson, Claude generates a pack from it:
- 1 LinkedIn post · 1 X thread · 1 Reel/Short script · YouTube notes (title/desc/thumbnail text/talking points).
Vishal personalizes + posts.

## Cadence (start small)
LinkedIn 3–4/week · X daily-ish · YouTube weekly live + ~1 edited video · Insta repurpose 2–3/week. Scale up only after the habit sticks.

## What Vishal should do differently
Proof>claims · engage>broadcast · consistency>perfection · learner-not-guru · one niche.
